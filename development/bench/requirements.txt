#
# Why every transitive dep is pinned:
#   - ragatouille 0.0.9 imports langchain.retrievers (the pre-0.3 API)
#     and breaks against langchain 0.3+
#   - colbert-ai 0.2.22 uses HF_ColBERT.all_tied_weights_keys which
#     transformers 5.x removed; needs transformers 4.x
#   - fast-pytorch-kmeans uses psutil without declaring it; pip's
#     resolver doesn't pull it in
#   - pip's resolver under a permissive constraint picks the newest
#     compatible-on-paper version, hitting all three bugs above.
#
# Result: a fresh `pip install ragatouille>=0.0.9 torch>=2.0` will
# silently produce a broken venv that fails on first index-build.
# This frozen file works.
#

#   bash scripts/setup-colbert.sh                  # recreates the venv
